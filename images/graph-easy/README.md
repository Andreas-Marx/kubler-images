## kubler/graph-easy

Run this [graph-easy][] image with:

    $ docker run -it --rm kubler/graph-easy graph-easy --help

The image also includes graphviz if you prefer to use it directly:

    $ docker run -it --rm kubler/graph-easy dot --help
